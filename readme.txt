Pour pouvoir uploader des vidéos de taille raisonnable sur le serveur, possible en temps qu'administrateur,
il faut changer la taille max de l'upload.
-ouvrir php.ini (dans xampp/) 
-changer les parametres 
upload_max_filesize = 10M
post_max_size = 10M
-rédémarrer le serveur Apache

Et ça fonctionne
